# Changelog

All notable changes to the "EventChain AI-PDS" project will be documented in this file.

## [v1.3.1] - Planned (Current Objective)
### Added
- **Dark Mode**: Native Tailwind based dark mode with persistence.
- **Multi-language**: Tamil and Hindi support (offline).
- **Accessibility**: ARIA labels, keyboard nav, WCAG AA contrast.

## [v1.2.0] - 2026-01-23 (Current Active Version)
### Changed
- **UI Rollback**: Reverted incomplete Dark Mode/i18n implementation to ensure stability.
- **Restored**: "Government-Grade" Light Mode UI.

### Added
- **System Status Bar**: Real-time monitoring of Blockchain, IPFS, AI.
- **Flow Tracker**: Visual step-by-step progress for PDS distribution.
- **Transaction Cards**: expandable cards with embedded QR codes.
- **Verification UI**: Official government-style verification seal and summary.
- **Authority Dashboard**: Visual analytics for Shop Trust and Regional Fraud.

## [v1.1.0] - 2026-01-22
### Added
- **AI Fraud Logic**: Enhanced synthetic data, heuristic boosting, and risk scoring.
- **Verification Logic**: Improved PDF layout and QR code JSON parsing.
- **Security**: Added integrity proofs (Merkle/Hash) to verification results.

## [v1.0.0] - Initial Release
### Added
- Core Blockchain (Hardhat/Solidity).
- AI Microservice (Python/FastAPI).
- Backend API (Node.js).
- Basic Frontend (React).
